# Chrome插件 博客文章内容提取导出

> fork from: https://github.com/fandsimple/chrome_copy

该插件目的是方便同学们笔记学习，仅供学习交流之用。



# 功能描述

- CSDN 博客文章
  - 导出为MarkDown
  - 导出为PDF
- cnblog 博客文章
  - 导出为MarkDown
- jianshu  博客文章
  - 导出为MarkDown
- Juejin
  - 导出为MarkDown



# 平台支持

- csdn
- 博客园
- 简书
- 掘金（有bug，在解决）
- https://www.dgstack.cn/该站下文章
